# AI Agents in Action Playground

Code companion for working through [AI Agents in Action](https://www.manning.com/books/ai-agents-in-action) by Micheal Lanham, using the [openai-agents](https://pypi.org/project/openai-agents/) Python SDK.

## Setup

```sh
uv sync
```

Set your OpenAI-compatible credentials:

```sh
export OPENAI_API_KEY=your-key
export OPENAI_BASE_URL=your-base-url
```

## Usage

```sh
uv run python ch02/01_first_agent.py
```
